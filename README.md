What is Americoin?
----------------

Americoin is a lite version of Bitcoin using scrypt as a proof-of-work algorithm.
 - 15 second block targets
 - subsidy halves in 250k blocks (~4 years)
 - 1 coin per block
 - 1 block to retarget difficulty
